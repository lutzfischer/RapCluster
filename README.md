# RapCluster

This project provides a **Flask backend** and a **React frontend** for clustering analysis and visualization.  
The webpage is generated by a **React app**.

## Setup Instructions

### Create and activate a Conda environment
```bash
conda create -n clustering_env python=3.9
conda activate clustering_env
```
### Install backend dependencies
```bash
pip install Flask Flask-Cors scikit-learn numpy pandas umap-learn hdbscan openpyxl matplotlib
```
### Run the Flask backend
```bash
cd /path/to/your/app/directory
python app.py
```
###  Install frontend dependencies
```bash
cd clustering-frontend
npm install react-plotly.js react-papaparse xlsx
npm install
```
###  Start the React frontend
```bash
cd clustering-frontend
npm start
```
